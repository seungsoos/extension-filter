# Extension Filter Frontend

파일 확장자 차단 관리 React 애플리케이션

## 기술 스택

- React 18.3
- Vite 6.0
- Axios

## 개발 서버 실행

```bash
npm run dev
```

http://localhost:3000 에서 확인 가능

## 프로덕션 빌드

```bash
npm run build
```

빌드된 파일은 `../src/main/resources/static` 에 생성됩니다.

## 주요 기능

- 고정 확장자 체크박스 관리 (7개)
- 커스텀 확장자 추가/삭제 (최대 200개)
- 실시간 API 연동
